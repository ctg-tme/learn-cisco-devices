‚ùå Incorrect: Direct Image Embed

This will NOT track analytics (bypasses the SPA):

<img src="https://ctg-tme.github.io/learn-cisco-devices/deployments/mtr-navigator/images/mtr_navigator_schedule_teams.png">

Why it doesn't work: Direct file access doesn't load the SPA, so no JavaScript runs and no analytics are captured.
